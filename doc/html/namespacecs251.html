Detailed Description

The namespace contains all the major variables and function definitions.

This is the one and only namespace in the code. This has all the variables and classes like dominos_t which have the main elemnet generation code and callback code of this project.

Callback functions are called when a event occurs due to the press of a key, mouse button, or a GUI element like a menu item is clicked or scroller is dragged. Some callbacks are triggered after a fixed intervals when a timer elapses.
